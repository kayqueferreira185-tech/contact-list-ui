# Contact List UI

Projeto de interface de lista de contatos desenvolvido com HTML e CSS, com foco em estrutura semântica, organização visual e responsividade.

## 🚀 Sobre o projeto

A aplicação simula uma lista de contatos com layout organizado, pensando em aplicações reais como:
- sistemas internos de empresas (CRM simples)
- agendas digitais corporativas
- dashboards administrativos
- ferramentas de organização de equipes e clientes

O objetivo do projeto foi praticar estruturação de layout e organização de componentes utilizando HTML semântico e CSS, simulando uma interface que poderia ser utilizada em ambientes profissionais.

## 💡 Aplicações no mundo real

Esse tipo de interface pode ajudar empresas e usuários em situações como:
- organização de contatos de clientes ou funcionários
- visualização rápida de informações importantes
- sistemas internos de gestão simples
- melhoria na produtividade ao centralizar dados em uma interface clara

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3

## 📂 Estrutura do projeto

- index.html → estrutura principal da página
- estilo.css → estilização da interface
- imagens/ → assets do projeto

## 📌 Aprendizados

Durante o desenvolvimento, foram praticados:
- HTML semântico
- organização de layout com CSS
- estruturação de componentes visuais
- boas práticas de organização de projeto
- pensamento voltado para aplicações reais de interface
